# Padrões para documentação do projeto

> Escreva sempre em português brasileiro.

- Use headings e subheadings para organizar o conteúdo.
- Use listas com marcadores ou numeradas para clareza.
- Use blocos de código para trechos de código.
- Inclua links para referências e leitura complementar.
- Mantenha a documentação sincronizada com as mudanças no código.
- Mantenha uma seção de créditos para bibliotecas de terceiros usadas no projeto.
